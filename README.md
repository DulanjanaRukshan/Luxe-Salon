## ‚ú® Features

### Frontend (Client)
- **Luxury UI/UX:** Glassmorphism effects, smooth Framer Motion animations, and parallax scrolling.
- **Appointment Booking:** Users can select services, dates, and book appointments directly.
- **Gallery:** Interactive masonry grid with a full-screen lightbox view.
- **Services Menu:** Categorized tabs (Hair, Skin, Spa, Bridal) with pricing.
- **Testimonials:** Client reviews carousel with verified badges.
- **FAQ Section:** Accordion-style expandable questions.
- **WhatsApp Integration:** Floating chat button for direct communication.

### Backend (Server)
- **API:** RESTful API built with Node.js & Express.
- **Database:** MongoDB Atlas (Cloud) integration for storing data.
- **Bookings:** Saves appointment requests securely.
- **Newsletter:** Captures and stores subscriber emails.
- **Security:** CORS enabled and Environment variables for sensitive data.
